При проведении эксперимента управление работой цифровой камеры во время
эксперимента осуществляется дистанционно с ПК, на котором должна быть
установлена программа управления камерой. Для цифровой камеры F1 существует
возможность дистанционного запуска с ПК (кабель USB в комплекте).

Порядок действий:

\begin{enumerate}

\item Скачать софт для управления камерой через USB можно по этой ссылке:
\url{http://support.casio.jp/download.php?cid=001\&pid=567}.
В архиве находятся 4 файла (см. рис.~\ref{fig:achiv}).

\item После этого нужно пропатчить цифровую камеру F1 до версии 2.0.
Для этого нужно на SD карту в корневой каталог поместить файл EX-F1.bin, включить камеру с флешкой.
Проверить версию можно одновременно зажав клавишу меню и клавишу включения  (см. рис.~\ref{fig:vers}).

\item После этого устанавливаете софт.

\item Подключить камеру через USB стандартным кабелем и в меню
(сбросится русский язык, для его установки можно, если нужно,
качать с сайта русскую версию):
Menu -> settings -> usb mode включаете "PC control"\ (в русской
версии будет называться ("pictbridge") (см. рис.~\ref{fig:usb}).

\item % На рисунке \ref{fig:CASIO} показан вид экрана с ярлыком программы.
После запуска программы экран ПК будет выглядеть так, как показано на рисунке~\ref{fig:start}.
Выбираете камеру и нажимаете <connect>.

\item  На рисунке \ref{fig:video} показан экран дисплея,
на которое выведено  окно программы с кнопкой <Movie> -- <ЗАПИСЬ ВИДЕО>,
а также изображение объекта с подключенной цифровой камеры.

\end{enumerate}

При проведении эксперимента необходимо соблюдать следующую последовательность
действий:

\begin{enumerate}

\item Подготовить план проведения эксперимента, включая таблицу частот переключения
магнитного поля.

\item Установить станину в требуемое вертикальное или горизонтальное положение

\item Закрепить на станине канал.

\item Ввести в канал исследуемый образец намагничивающегося упругого тела и выставить его
в стартовую позицию, ориентируясь по меткам на мерной линейке. Обычно это
соответствует середине одного из сердечников, причем сердечник выбирается из расчета,
чтобы вся длина образца помещалась между катушками электромагнитной системы.

\item Установить цифровую камеру, подключить ее к компьютеру, задать режим съемки и
сфокусировать ее на образец, чтобы кадр захватывал и стартовую и конечную позицию
образца. Примечание: Кадр должен захватывать и метку с указанием значения частоты,
устанавливаемую рядом с каналом, чтобы в дальнейшем знать частоту переключения поля
в эксперименте.

\item На панели БУ установить переключатель паузы в нужное положение от 2 до 7 тактов
переключения магнитного поля, чтобы цикл деформации 2-х соседних сегментов образца
был закончен.

\item Задать режим работы БУ (см. пп. 1~-- 8 на \pageref{ordung_BU} странице).

\item Запустить цифровую камеру с ноутбука.

\item Запустить генерацию магнитного поля (см. п. 9 на \pageref{ordung_BU} странице).

\item По завершении этапа эксперимента выключить цифровую камеру.

\item Выключить коммутатор БУ кнопкой Zero и перевести генератор в режим ручного
однократного пуска (см. п. 10~-- 11 на \pageref{ordung_BU} странице).

\item Для выполнения следующего этапа эксперимента при другой частоте переключения
магнитного поля надо перенести образец на стартовую позицию, установить новую метку
для частоты, задать новый режим работы БУ (см. пп. 5~-- 6 на \pageref{ordung_BU}
странице). Далее для проведения последующих этапов эксперимента пп. 5~-- 7 и 9~-- 11
повторяются.

\end{enumerate}
